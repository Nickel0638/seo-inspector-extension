# SEO Inspector Extension

## Встановлення
1. Отримай API ключ на [Google Cloud Console](https://console.cloud.google.com)
2. Увімкни PageSpeed Insights API
3. В файлі `popup.js` замість `YOUR_API_KEY_HERE` встав свій ключ
4. Завантаж розширення в Chrome через `chrome://extensions/` → Load unpacked
